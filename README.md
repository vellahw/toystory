하나의 웹 어플리케이션에서 다양한 즐거움을 만족 시키고 싶은 마음에서 시작된 Spring Framework 기반 MVC 패턴의 장난감 쇼핑몰 토이스토리입니다. <br> 
토이스토리에서는 피규어 뿐만 아니라 취미와 취향 찾기를 위한 다양한 DIY 키트, 문구/팬시 등을 만나볼 수 있습니다.<br>
